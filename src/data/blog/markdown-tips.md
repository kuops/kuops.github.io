Markdown 是最常用的轻量级标记语言。除了基本语法外，许多扩展功能可以显著提升写作体验。

## 基本语法回顾

- **粗体**：**加粗文字**
- _斜体_：_斜体文字_
- `行内代码`：`code`

## 表格

| 功能       | 支持度 | 说明               |
| ---------- | ------ | ------------------ |
| GFM 表格   | ✅     | 标准 Markdown 扩展 |
| 对齐       | ✅     | 支持左中右对齐     |
| 合并单元格 | ❌     | 原生不支持         |
