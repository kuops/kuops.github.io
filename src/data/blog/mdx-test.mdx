这篇文章用于验证博客的 MDX 支持是否正常工作。MDX 允许在 Markdown 中混合使用 JSX 语法。

## 普通 Markdown

MDX 完全兼容标准 Markdown 语法：**加粗**、_斜体_、`行内代码`、[链接](https://astro.build/)。

## 变量和表达式

MDX 可以在 frontmatter 下方定义变量，然后在内容中引用：

export const siteName = "My Blog";
export const year = new Date().getFullYear();
export const posts = ["Hello World", "MDX 测试", "Astro 指南"];

当前站点：**{siteName}**，当前年份：**{year}**。

export const product = 2 * 3;

数学运算：2 × 3 = {product}，数组的长度是 {posts.length}。

## 条件渲染

export const showAdvanced = true;

{showAdvanced ? (

  <div style={{ padding: "1rem", border: "1px solid var(--color-border)", borderRadius: "8px", margin: "1rem 0", color: "var(--color-foreground)" }}>
    <strong>高级内容已启用</strong> — 这段内容通过 JSX 条件渲染显示。
  </div>
) : (
  <p>高级内容已隐藏。</p>
)}

## 列表渲染

用 JSX 的 `map` 动态生成列表：

<ul style={{ paddingLeft: "1.5rem" }}>
  {posts.map((post, i) => (
    <li key={i}>
      {i + 1}. {post}
    </li>
  ))}
</ul>

## 样式化组件
