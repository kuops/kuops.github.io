## 什么是 Astro

Astro 是一个用于构建**内容驱动网站**的 Web 框架。它于 2021 年发布，迅速成为最受欢迎的静态站点生成器之一。

### 核心理念

Astro 的设计围绕三个核心理念：

1. **以内容为中心** — Astro 认为 HTML 是最好的内容载体
2. **零 JavaScript 默认** — 默认不发送 JS 到客户端，页面加载极快
3. **岛屿架构** — 需要交互的组件可以独立 hydrate，互不干扰

### 与其他框架的对比

| 特性     | Astro     | Next.js | Nuxt    |
| -------- | --------- | ------- | ------- |
| 默认输出 | 静态 HTML | SSR/SSG | SSR/SSG |
| JS 体积  | 接近零    | 中等    | 中等    |
| 学习曲线 | 低        | 中      | 中      |
| 框架绑定 | 无        | React   | Vue     |

## 为什么选择 Astro

如果你在构建以下类型的网站，Astro 是绝佳选择：

- **博客** — 纯内容展示，不需要复杂交互
- **文档站点** — Markdown 驱动，搜索友好
- **营销页面** — 加载速度直接影响转化率
- **作品集** — 展示项目，注重视觉效果

## 环境准备

在开始之前，确保你的开发环境已就绪：

- **Node.js** 18.14.1 或更高版本
- **编辑器** — 推荐VS Code + Astro 官方插件
- **终端** — 任意终端工具

验证 Node.js 版本：

```bash
node --version
# v18.x.x 或更高
```
