## Astro 组件基础

Astro 组件是 `.astro` 文件，由两部分组成：

### 组件结构

```astro
---
// Frontmatter：服务端运行的 JavaScript/TypeScript
const title = "Hello Astro";
---

<!-- Template：HTML 模板 -->
<h1>{title}</h1>
<p>这是我的第一个 Astro 组件。</p>

<style>
  h1 {
    color: purple;
  }
</style>
```

### Props 传参

组件可以接收外部传入的属性：

```astro
---
interface Props {
  title: string;
  count?: number;
}

const { title, count = 0 } = Astro.props;
---

<h2>{title}</h2>
<p>计数：{count}</p>
```

使用时传入 props：

```astro
<Card title="文章标题" count={5} />
```

## 页面路由

`src/pages/` 目录下的文件自动映射为路由：

```
src/pages/
  index.astro        → /
  about.astro        → /about
  blog/
    index.astro      → /blog
    [slug].astro     → /blog/:slug
```

### 动态路由

使用 `[slug]` 语法创建动态路由：

```astro
---
export function getStaticPaths() {
  return [
    { params: { slug: 'hello-world' } },
    { params: { slug: 'second-post' } },
  ];
}

const { slug } = Astro.params;
---

<h1>文章：{slug}</h1>
```

## 布局组件

布局是可复用的页面骨架：

```astro
---
// src/layouts/Base.astro
interface Props {
  title: string;
}

const { title } = Astro.props;
---

<html>
  <head>
    <title>{title}</title>
  </head>
  <body>
    <nav>导航栏</nav>
    <main>
      <slot />
    </main>
    <footer>页脚</footer>
  </body>
</html>
```

`<slot />` 是页面内容的插入点。

## 样式处理

Astro 支持多种样式方案：

### Scoped CSS

```astro
<style>
  /* 只作用于当前组件 */
  .card {
    padding: 1rem;
    border-radius: 8px;
  }
</style>
```

### 全局 CSS

```astro
<style is:global>
  body {
    margin: 0;
  }
</style>
```

### CSS 框架

可以通过集成使用 Tailwind CSS 等框架：

```bash
npx astro add tailwind
```
