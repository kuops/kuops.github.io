## 创建新项目

使用 Astro 官方的创建向导来初始化项目：

```bash
npm create astro@latest
```

向导会询问你几个问题：

1. **项目名称** — 例如 `my-blog`
2. **模板选择** — 推荐选择 `minimal` 从零开始
3. **是否安装依赖** — 选择 Yes
4. **是否初始化 Git** — 选择 Yes
5. **是否使用 TypeScript** — 推荐选择 Strict

## 项目结构

创建完成后，你会得到如下目录结构：

```
my-blog/
├── public/          # 静态资源（图片、字体等）
├── src/
│   ├── components/  # UI 组件
│   ├── layouts/     # 页面布局
│   ├── pages/       # 页面路由
│   └── styles/      # 全局样式
├── astro.config.mjs # Astro 配置
├── package.json
└── tsconfig.json
```

### 关键目录说明

- `src/pages/` — 文件即路由，`index.astro` 对应首页
- `src/components/` — 可复用的 UI 组件
- `src/layouts/` — 页面骨架模板
- `public/` — 不会被处理的静态文件

## 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:4321` 即可看到默认页面。

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 构建生产版本 |
| `npm run preview` | 预览构建结果 |

## 配置文件

`astro.config.mjs` 是 Astro 的核心配置文件：

```js
import { defineConfig } from 'astro/config';

export default defineConfig({
  site: 'https://example.com',
  output: 'static',
});
```
